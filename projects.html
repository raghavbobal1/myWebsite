Slick Notes

An iOS note taking application

• A note-taking app built on x-code using swift.
• User can sort notes by title or by date/time that were created..
• Information on where was the note taken (Location) is also captured as part of the note.
• Supports Multimedia like photos and voice recording.
• User can change the category of the note as well.
• User can record audio and attach it to the note or use a pre-recorded sound on the device.
• Application allows the user to see the location on a map for every note that was taken.

Slick Notes

An android note taking application

• A note-taking app built on android studio using java.
• User can sort notes by title or by date/time that were created.
• User can search for specific note by title or by keyword that may be contained in a note.
• User can take a picture (or use a picture that was previously taken) and store it as part of a note.
• Information on where was the note taken (Location) is also captured as part of the note.
• Application allows the user to see the location on a map for every note that was taken.
• User can record audio and attach it to the note or use a pre-recorded sound on the device.
• User can change the subject/category of a note i.e. move a note from one subject/category to another.
